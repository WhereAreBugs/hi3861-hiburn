## Hi3861-HiBrun-tool

### 简介

一个跨平台的Hi3861的hiburn协议的烧录脚本。<br>

### 使用

用法很简单，只需要传入要烧录的文件路径即可，支持将文件拖拽到python文件上。<br>
使用-h可以查看帮助
